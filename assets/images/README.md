# Image assets

Drop your image files into this folder. Below is the list of filenames the
HTML expects. You can either:

- Rename your files to match these names (easiest), **or**
- Tell me what your filenames are and I'll update the HTML.

## Required

| Filename                | Used on               | Notes                                                 |
| ----------------------- | --------------------- | ----------------------------------------------------- |
| `logo.svg` or `logo.png`| every page (header)   | Transparent background. SVG preferred.                |
| `favicon.png`           | every page (browser tab) | 512x512 ideal, square, will be auto-scaled.        |
| `hero.jpg`              | `index.html`          | Wide landscape (e.g. 1920x1080). The big top image.   |
| `about-jan.jpg`         | `about.html`          | Optional portrait of Jan. Square or 4:5 works best.   |

## Gallery (project photos)

Name them `project-01.jpg`, `project-02.jpg`, … through `project-20.jpg`.
The HTML currently links to 12 of them; add or remove as you like and I'll
adjust. Landscape orientation (4:3 or 3:2) looks best in the grid.

## Image tips for SEO + speed

- JPG for photos, PNG/SVG for the logo.
- Keep each photo under ~500 KB if you can. I can compress them later if needed.
- Aim for max 2000px on the longest edge — anything bigger is wasted on the web.
